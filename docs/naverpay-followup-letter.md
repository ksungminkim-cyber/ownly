## 발송 후 후속 조치

1. **회신까지 평균 5~10 영업일** 소요
2. 추가 질문이 오면 빠르게 답변 (24시간 이내가 이상적)
3. 만약 또 거절되면:
   - 거절 사유 정확히 파악
   - 위 자료에서 어떤 부분이 부족했는지 확인
   - 1:1 통화 요청 (1588-3819)
4. 가맹 승인 시:
   - 네이버페이 비즈센터 → API 키 발급
   - `/api/billing/naver/*` 라우트 구현
   - 체크아웃 페이지의 "심사중" 뱃지 제거
   - 본격 활성화
