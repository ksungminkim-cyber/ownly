## 첨부할 PDF 만드는 방법

1. `docs/ownly-saas-overview.html` 파일을 **크롬 브라우저**로 열기
   - VS Code에서 우클릭 → "Reveal in File Explorer" → 더블클릭
   - 또는 파일 경로를 크롬 주소창에 붙여넣기
2. **Ctrl + P** (인쇄)
3. 대상: **"PDF로 저장"** 선택
4. 여백: **기본** / 옵션: **"배경 그래픽" 체크**
5. 저장 → `ownly-saas-overview.pdf` 파일 생성

이 PDF를 메일에 첨부하시면 됩니다.
