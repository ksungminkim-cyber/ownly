## 2. 서비스 정의

**온리(Ownly)는 임대인을 대상으로 한 클라우드 기반 업무 관리 소프트웨어(SaaS)입니다.**

### 2.1 서비스의 본질

```
[ 본 서비스 ]                          [ 본 서비스가 아닌 것 ]
✓ 임대인용 업무 자동화 SaaS            ✗ 부동산 매매 중개
✓ 클라우드 소프트웨어 사용권 판매       ✗ 부동산 임대 중개
✓ 월 구독료 수취                       ✗ 부동산 매물 광고
✓ 업무 관리 도구 제공                  ✗ 부동산 거래 수수료 수취
                                       ✗ 보증금·임대료의 직접 처리
```

### 2.2 비즈니스 모델

- **수익 구조**: 월/연 SaaS 구독료 (단일 수익원)
- **결제 대상**: 임대 관리 소프트웨어 사용권
- **거래 형태**: 전자상거래 → 디지털 상품(SaaS 구독권) 판매

이는 더존비즈온(회계 SaaS), 자비스앤빌런즈(세무 SaaS), 슬랙(협업 SaaS)과
**동일한 SaaS 비즈니스 모델**이며, 단지 타겟 사용자 segment가 "임대인"입니다.

## 7. 부동산 관련 키워드 사용에 대한 해명

당사 사이트 일부에 "부동산", "임대" 등 키워드가 노출되는 것은
**SaaS 사용자(임대인)와의 소통을 위한 것**일 뿐, 사업의 성격이 부동산 거래임을 의미하지 않습니다.

비교:
- **세무 SaaS** → "세금 신고", "부가세" 같은 키워드 사용 (그러나 세무 대행 X)
- **회계 SaaS** → "재무제표", "결산" 같은 키워드 사용 (그러나 회계 대행 X)
- **임대 관리 SaaS** → "임대료", "공실" 같은 키워드 사용 (그러나 부동산 거래 X)

이는 **"타겟 사용자에게 적합한 마케팅 언어"** 일 뿐이며, 제공 상품의 본질은
모두 **"업무 관리용 클라우드 소프트웨어"** 입니다.
