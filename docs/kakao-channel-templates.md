## 6. 채널 운영 팁

1. **첫 인사 메시지 길지 않게** — 카톡은 짧은 메시지가 더 친근
2. **링크는 풀URL** — 카톡은 단축 URL 미지원 (생성됐던 것이 차단되는 경우 있음)
3. **이모지 적당히** — 너무 많으면 읽기 불편
4. **빠른 답장 단축어** 잘 활용 → 응대 시간 단축
5. **VIP 친구** 등록 → 결제 유저는 우선 응대
6. **자동 응답 vs 수동 응답** 구분 명확하게 — "자동" 표시 안 하면 사용자 혼란

## 7. 알림톡 발송 (PRO 플랜용 — 카카오 비즈메시지 별도 신청 필요)

알림톡은 일반 채널 메시지와 달리:
- 정보성 메시지 한정 (광고 금지)
- 사전 검수된 템플릿만 발송 가능
- 1건당 ~10원 발송비

**활성화 방법**:
1. 카카오 비즈센터 → 비즈메시지 → 발신프로필 등록
2. 알림톡 템플릿 등록 (3~5일 검수)
3. 온리 백엔드에 발신 API 연동 (PRO 플랜에서 제공 예정)
